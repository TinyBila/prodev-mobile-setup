## Mobile Development Setup
This project sets up the mobile development environment for React Native using the Expo Framework.
Expo Go allows developers to preview and test React Native applications directly on physical devices, without needing an emulator.

## Tools & Requirements
Node.js LTS – for running React Native scripts.

VS Code – for editing and managing the project files.

Operating System: Windows / macOS / Linux.

Expo Go App – for testing React Native apps on a physical device.
